# Quiz Application

A fun and interactive quiz game built in Python that 
tests your knowledge through multiple choice questions.
Built as part of my internship training at 
SkillInfyTech IT Solutions.

## Project Overview
I always enjoyed playing quiz games so building one 
from scratch was really exciting. This project taught
me how to control program flow, validate user input 
and keep track of a score — all using basic Python.

## Features at a Glance
- Displays questions one at a time
- 4 answer options for each question
- Instantly tells you if your answer is right or wrong
- Keeps a running score throughout the game
- Shows your final result at the end

## Technologies Used
- Python 3
- No external libraries needed

## Getting Started
Step 1 — Make sure Python 3 is installed
Step 2 — Download or clone this repository
Step 3 — Open terminal and run:
         python quiz.py

## Skills I Picked Up
Through this project I got hands on experience with
using loops to repeat questions, writing conditional
logic to check answers, handling invalid inputs and
building a smooth interactive experience in the 
terminal.
